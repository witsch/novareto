nva.chemiedp Installation
-------------------------

To install nva.chemiedp using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``nva.chemiedp`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        nva.chemiedp
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
